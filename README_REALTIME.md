# 实时监控系统使用说明

## 快速启动

### 1. 启动实时监控服务器

双击运行：
```
start_server.bat
```

或者在命令行运行：
```bash
cd C:\Users\Administrator\Desktop\quant-showcase
python realtime_server.py
```

### 2. 访问监控页面

打开浏览器访问：
```
http://localhost:8088
```

或者访问在线版本（需要先运行服务器）：
```
https://quant-showcase-lzl.netlify.app
```

## 工作原理

```
┌─────────────────────────────────────────────────────────┐
│                    您的电脑                              │
│  ┌─────────────────┐      ┌─────────────────┐          │
│  │ realtime_server │ ───→ │ monitor.html    │          │
│  │ (端口8088)      │ API  │ (浏览器)        │          │
│  └─────────────────┘      └─────────────────┘          │
│           │                      ↑                     │
│           ↓                      │                     │
│  ┌─────────────────┐      ┌─────────────────┐          │
│  │ astock_analyzer │      │ Netlify托管     │          │
│  │ (数据源)        │      │ (静态文件)      │          │
│  └─────────────────┘      └─────────────────┘          │
└─────────────────────────────────────────────────────────┘
```

## 数据刷新频率

| 数据类型 | 刷新频率 | 说明 |
|----------|----------|------|
| 市场数据 | 30秒 | 上涨/下跌/涨停/跌停 |
| 情绪指标 | 30秒 | L1-L5情绪等级 |
| 指数涨跌 | 30秒 | 上证/深成指/创业板 |
| 持仓信息 | 30秒 | 当前持仓股票 |
| 交易记录 | 30秒 | 今日交易明细 |

## 停止服务器

在服务器窗口按 `Ctrl+C` 即可停止。

## 常见问题

### Q: 服务器启动后访问不了？
A: 检查端口8088是否被占用，或者防火墙是否阻止。

### Q: 数据没有更新？
A: 确保astock_analyzer系统正在运行，并且有新的数据生成。

### Q: 外网可以访问吗？
A: 本地服务器只能在局域网内访问。如需外网访问，请使用Netlify版本。
